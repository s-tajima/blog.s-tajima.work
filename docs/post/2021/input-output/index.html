アウトプット

登壇

今年も引き続きオンライン登壇のみ。来年はそろそろ物理登壇もしていきたい。

  • 日々変わりゆく技術トレンドの中でスタートアップ4社が技術選定について語ります
  • AWS フィンテック 特別セミナー フィンテック 企業が見据える近未来の技術トレンド

インタビュー

  • Flatt Securityさんの事例紹介インタビュー
  • AWS Startup ブログの事例紹介インタビュー

ブログ

TechBlogは例年どおりなんですが、今年はnoteにてフィードバック・評価制度を紹介する記事も書いてみました。 会社で働く上で無関係な人はほとんどいないテーマだったためか、かなり多くの人に読んでもらえたみたいでよかったです。

  • AWS Lambdaの汎用的なエラー通知を実現するLapperというツールをつくりました
  • 人事部無しで100人を超えたFinatextが実践するフィードバック・評価制度
  • AWSでクロスリージョン・クロスアカウントのDisaster Recoveryシステムを構築した話
  • AWS セレクトコンサルティングパートナーとISVパートナーパスの同時認定までの道のり
  • Platform Teamとしての上場までの約3年を振り返る
  • Platform Teamはなぜ “インフラ” ではなく “プラットフォーム” を冠しているのか

資格

せっかく金融スタートアップで働いているので、金融関連の資格はとっておこうということで少額短期保険募集人と一種外務員の資格を取りました。 今はOSCPに続きOSWEの取得に向けて絶賛勉強中です。
